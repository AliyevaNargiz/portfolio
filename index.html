My Projects

CANSAT Satellite System and Ground Station

Used: C/C++ (Arduino), C# (.NET Backend), HTML, CSS, JavaScript

The CanStellar project is a model satellite system that won 1st place at CANSAT Azerbaijan 2023 and represented the country at the Global CANSAT US 2024. It includes an Arduino-based satellite that collects real-time altitude and GPS data, transmits it via XBee, and logs it to an SD card. The data is processed by a .NET backend and visualized through a live web-based ground station.

Restaurant Ordering System

Used: Java (Spring Boot), Gradle, Node.js, React.js, Material-UI, PostgreSQL, H2 Database, HTML, CSS, JavaScript

This web-based system streamlines in-restaurant ordering with QR code table scanning, a responsive digital menu, real-time order tracking, and cart functionality. Built using React for the frontend and either Spring Boot (Java) or Express (Node.js) for the backend, with database support via H2 or PostgreSQL, it offers a complete end-to-end ordering experience.
