WEEK 8

Students should be able to:

  1. add jQuery to the website using either a downloaded library or included from a CDN.
  2. use different jQuery selectors.
  3. handle events using jQuery functions.
  4. use .on() function to register multiple handlers to different events
  5. practice setTimeout() and setInterval() functions
  6. use jQuery to access and manipulate CSS style and HTML attributes of an element
  7. create, add, and remove HTML elements into the DOM tree using jQuery
  8. using .on() function will help with registering listeners for dynamically created elements as well.
  9. e.g., Try the following:$(parent).on(eventType, selector, callback)
  10. add, remove, and toggle classes of an element using jQuery
  11. use AJAX to send GET/POST requests, receive responses, and process the data
  12. start a JSON server to fake theREST API without writing any backend code
  13. use Vanilla JS or jQuery to send/receive HTTP requests/responses to the fake JSON server
  14. use alternatives of AJAX, such as Fetch API or Axios HTTP client.
